# todo-py-django

## Requirements

- Python 3.10
- Poetry

## Development

```
git clonr https://github.com/anshulxyz/todo-py-django
cd todo-py-django
poetry shell
poetry install
```

Run test
```
make test
```

Run app

```
make run
```
